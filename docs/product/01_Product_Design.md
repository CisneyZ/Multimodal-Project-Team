# MPT Platform v0.1 产品设计

## 1. 产品定位

MPT Platform 是面向影视与 AI 项目团队的内部人才筛选平台。通过简历解析、人才画像、项目规则匹配、硬性条件筛选和可解释评估报告，帮助招聘人员与项目负责人快速判断候选人是否满足要求、最适合哪个项目、推荐依据是什么、还缺哪些信息。

## 2. v0.1 业务边界

仅实现“高阶影视人才筛选”，支持两个内置项目：

1. RL叙事剪辑
2. Caption-Diff Caption

数据结构必须支持未来新增项目，但第一版不做复杂规则编辑器。

## 4. 核心闭环

```text
登录
→ 新增候选人
→ 上传或粘贴简历
→ 解析简历
→ 人工确认人才画像
→ 同时匹配两个项目
→ 硬性条件筛选
→ 生成评估报告
→ 保存至人才库
→ 更新招聘状态
```

## 5. 功能模块

### 首页

展示候选人总数、待确认、已评估、推荐人数、项目数量和最近筛选记录。

### 项目管理

展示两个内置项目的要求、评分维度、硬性条件和匹配人才。

### 候选人管理

支持新增、编辑、查看、软删除、重新解析、重新评估、搜索筛选和招聘状态流转。

### 简历录入与解析

支持 PDF、DOCX、TXT 或文本粘贴中的可行路径；解析失败时必须允许手工录入。

### 人才画像

基础信息、影视能力、数据任务经验、工作条件、缺失信息和字段来源均可编辑并保存。

### 项目匹配

同一候选人同时获得两个项目的独立结果，包括总分、推荐等级、维度得分、命中证据、未满足项、缺失信息和硬性条件状态。

### 评估报告

报告必须保存并支持再次打开，包含候选人摘要、项目对比、推荐依据、证据、风险、缺失信息、追问、下一步、分析模式、规则版本和时间。

## 6. AI、规则与人工

```text
AI：信息提取、摘要、证据定位、报告润色
规则：评分、硬性条件、推荐等级、阻断
人工：修正、补充、最终业务确认
```

禁止让大模型自由决定最终分数与硬性结论。

## 7. 成功标准

- 一份真实简历可完整跑通；
- AI 失败后仍可完成筛选；
- 评分可解释；
- 硬性条件可以覆盖高分；
- 候选人、筛选结果和报告刷新后仍存在；
- 人才库可以再次打开历史记录。
