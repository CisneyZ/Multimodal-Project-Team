# MPT Platform v0.1 用户流程

## 主流程

```text
管理员登录
→ 首页
→ 点击开始人才筛选
→ 上传文件或粘贴简历
→ 文件解析
→ AI辅助结构化或规则提取
→ 人才画像确认
→ 人工修改并保存
→ 同时匹配两个项目
→ 执行硬性条件
→ 生成并保存报告
→ 进入人才库
```

## 异常流程

### 文件解析失败

```text
显示失败原因
→ 保留候选人草稿
→ 允许粘贴文本
→ 允许手工填写
```

### AI 失败

```text
提示已切换规则模式
→ 规则提取
→ 人工确认
→ 正常评分和报告
```

### 信息不足

```text
保存缺失字段
→ 硬性条件显示信息缺失
→ 最终等级不超过待补充信息
```

### 硬性条件失败

```text
保留维度得分
→ 显示阻断条件
→ 最终结论为条件不满足
```
