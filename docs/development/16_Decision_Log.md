# MPT Platform v0.1 决策记录

- 正式品牌：MPT Platform / Multimodal Project Team Platform / MPT平台；
- 旧 Star.X/StarX 不再用于展示，旧本地存储键可保留兼容；
- v0.1 只做高阶影视人才筛选；
- 主闭环不包含动态试题；
- 内置 RL叙事剪辑和 Caption-Diff Caption；
- AI 不负责最终打分；
- 使用模块化单体；
- 优先复用现有数据库；
- UI 方向为 Dify 信息架构 + Apple 视觉语言；
- AI 失败必须 fallback；
- 候选人、画像、结果和报告必须持久化。
