# MPT Platform 版本路线图

## v0.1

高阶影视人才筛选：候选人、简历、画像、项目匹配、硬性筛选、报告、人才库。

## v0.2

飞书协同、审核卡片、招聘状态、批量导入导出。

## v0.3

项目试题、试标记录、人工评分、多评委与面试记录。

## v0.4

Prompt 质检、视频质检、素材检索与知识库。

## v0.5

Agent Registry、Tool Registry、Model Profiles、Task Runtime、权限与审计。

## v1.0

多业务模块、Agent、Workflow、飞书、多模型、多模态内容理解与企业级部署。

原则：每一版只解决一个明确问题，不提前制造大量空入口。
