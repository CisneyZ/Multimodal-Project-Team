# MPT Platform v0.1 演示数据与验收

## 案例 A：RL 高匹配

```text
林沐，本科毕业于某电影学院影视摄影与制作专业，3年剪辑经验。
负责品牌短片、纪录片和网剧的粗剪、精剪、节奏调整与基础调色。
熟悉镜头语言、构图、影调和叙事节奏，拥有可提供的作品集。
可全职到岗，可稳定参与12个月，愿意试标。
```

预期：RL 明显高于 Caption，硬性条件全部通过。

## 案例 B：Caption 高匹配

```text
周叙，本科毕业于某传媒大学戏剧影视文学专业。
有短剧编剧、导演助理、视频文字标注与内容质检经验。
能够输出结构化画面、动作、镜头与时间变化描述，文字表达准确。
拥有短剧、广告脚本和标注项目记录，可全职，愿意试标。
```

预期：Caption 明显高于 RL，硬性条件全部通过。

## 案例 C：硬性失败

```text
王简，市场营销专业，从事销售工作1年。
无剪辑、编剧、导演、视频标注或视频质检经验，无影视作品。
只能兼职，最多参与1个月，不愿意试标。
```

预期：两项目均不推荐，明确显示阻断原因。

## 人工修正演示

将案例 B 的全职状态改为未知，结果应变为待补充信息；再改为是，恢复推荐。

## AI 故障演示

关闭 AI 后加载案例 A，系统提示进入规则模式，仍可完成评分与报告。
