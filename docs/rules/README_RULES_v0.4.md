# MPT 高阶影视人才画像匹配规则 v0.4

## v0.1 最终范围

```text
简历录入
→ 人才等级判断
→ 人才画像
→ 三项目匹配
→ 分析报告
```

当前不做：

- 试标题与试标；
- 培训表现；
- L3/L4场地；
- 入项审批；
- 最终录用判断。

## 三个项目

1. SFT 多模态
2. Caption-Diff Caption
3. 影视剧创作意图

## 全局人才门槛

候选人必须满足中级或高级任意一档：

```text
SENIOR / INTERMEDIATE → 进入项目匹配
REJECT → 不进入项目匹配
MANUAL_REVIEW → 信息补充或人工复核
```

## 关键原则

- 人才等级与项目匹配分数分开；
- 同一专业在不同项目可以得到不同结论；
- 未提及不等于不通过；
- AI负责提取，规则引擎负责判断；
- 作品、岗位、职责与产出决定证据强度；
- 试题与场地信息保留为未来扩展，不影响v0.1结果。
